Main():
    1. Initialize isContinuingService and isUsingMenu to true
    2. Load bank account information for all holders into memory
    3. Display welcome message
    4. While isContinuingService is true:
        a. Display prompt to sign into an account
        b. Read user response to continue service
        c. If user chooses not to continue, exit loop
        d. Display prompt to enter username
        e. Read username input
        f. Check if username exists in the database
        g. If username does not exist:
            i. Display option to create a new bank account
            ii. If user chooses to create a new account:
                - Call createNewBankAccount() method
                - Continue to the next iteration of the loop
            iii. Otherwise, restart the loop
        h. If username exists:
          1. Get the associated bank account object
          2. Display prompt to enter password
          3. Read password input
          4. Check if the entered password is correct
          5. If password is incorrect:
              i. Display option to retry entering password
              ii. If user chooses to retry, repeat steps j to l
              iii. If user chooses not to retry, restart the loop
          6. If password is correct:
              i. While isUsingMenu is true:
                  a. Call presentOptionMenu() method to present menu options to the user
                  b. If user chooses to sign out, exit the loop
    5. Display farewell message
    6. Exit program

Method presentOptionMenu(userBankAccount):
    1. Display option menu
    2. Read user's menu option choice
    3. Based on the choice, call the corresponding method:
        a. Deposit: call handleDeposit(userBankAccount)
        b. Withdrawal: call handleWithdrawal(userBankAccount)
        c. Transfer: call handleTransfer(userBankAccount)
        d. Account Statistics: call viewBankAccountStatistics(userBankAccount)
        e. Recent Transactions: call viewRecentBankAccountTransactions(userBankAccount)
        f. Sign-Out: set isUsingMenu to false

Method createNewBankAccount():
    1. Display prompt to choose a username
    2. Read username input
    3. Check if the username already exists
    4. If username exists, repeat steps 1 to 3
    5. Display prompt to choose a password
    6. Read password input
    7. Create a new bank account with the provided username and password
    8. Display success message

Method viewBankAccountStatistics(userBankAccount):
    1. Display account statistics for the given bank account

Method viewRecentBankAccountTransactions(userBankAccount):
    1. Display recent transactions for the given bank account

Method handleDeposit(userBankAccount):
    1. Display account options for the user to choose
    2. Read user's account choice
    3. Display prompt to enter deposit amount
    4. Read deposit amount input
    5. Deposit the specified amount into the chosen account
    6. Display transaction result

Method handleWithdrawal(userBankAccount):
    1. Display account options for the user to choose
    2. Read user's account choice
    3. Display prompt to enter withdrawal amount
    4. Read withdrawal amount input
    5. Withdraw the specified amount from the chosen account
    6. Display transaction result

Method handleTransfer(userBankAccount):
    1. Display account options for the user to choose 'from' account
    2. Read user's 'from' account choice
    3. Display account options for the user to choose 'to' account
    4. Read user's 'to' account choice
    5. Display prompt to enter transfer amount
    6. Read transfer amount input
    7. Transfer the specified amount from the 'from' account to the 'to' account
    8. Display transaction result
